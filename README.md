A multi-restaurant loyalty and rewards API. This system allows customers to earn rewards for their repeat business and provides restaurant administrators with the tools to manage their loyalty programs.

Built with **Laravel**, this backend is designed to be flexible, scalable, and easy to integrate with any front-end application (web, mobile, or POS).

---

## ‚ú® Features

### For Customers
*   **Track Progress:** Customers can view their current standing for rewards at every restaurant they visit.
*   **Earn Rewards:** Automatically earn rewards, like a free meal after a set number of visits.
*   **Universal Profile:** A single customer profile can be used across all participating restaurants.

### For Restaurant Admins
*   **Configure Rewards:** Admins can set the criteria for rewards (e.g., "a free meal after 5 visits").
*   **Manage Restaurants:** Onboard new restaurants and manage their specific loyalty programs.
*   **Customer Management:** View customer progress and reward history.
